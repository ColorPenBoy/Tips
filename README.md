# Tips
记录一些小知识点

## Tip1:

#### `const`的作用：
* 限制类型
* 仅用来修饰右边变量
* 被`const`修饰的变量只读

#### `static`的作用：
1、修饰局部变量

* 延长局部变量生命周期
* 只生成一次内存，只初始化一次
* 改变局部变量的作用域

2、修饰全局变量

* 只能在本文件访问
* 避免重复定义全局变量

#### `extern`的作用
先在当前文件中查找，如果没有找到，才去其它文件中查找

#### `static` + `const` : 声明一个只读静态变量
#### `extern` + `const` : 定义一个多个文件共同使用的全局变量
